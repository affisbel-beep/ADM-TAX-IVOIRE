RESUME DU PROJET
═══════════════════════════════════════════════════════════════════════════════════════

NOM : ADM Tax Ivoire
SLOGAN : "Conduisez malin, gagnez plus"

PROBLEME : Les chauffeurs de Gbaka a Abidjan perdent 2-3h/jour dans les 
embouteillages, subissent des amendes, et ne savent pas calculer leur 
rentabilite. Les collectivites (Mairie, Etat) perdent des milliards en 
recettes non tracees.

SOLUTION : Application mobile pour chauffeurs qui :
- Optimise les trajets (carte offline, alertes police, zones chaudes)
- Taxe automatiquement chaque trajet (50 FCFA)
- Repartit transparentement : 25F Etat, 10F Mairie, 10F Syndicat, 
  5F Police, 5F Operateur

MODELE ECONOMIQUE :
- Fonctionne 100% hors ligne (realite africaine)
- Sync automatique quand connexion disponible
- Paiement via Orange Money, Wave, MTN MoMo, ou agent physique
- Commission syndicat pour adoption massive

COMMUNE PILOTE : Yopougon
PHASE 1 : 500 chauffeurs en 6 mois
INVESTISSEMENT : ~8 000 000 FCFA
RENTABILITE : A partir de 800 chauffeurs actifs

RAPPELS IMPORTANTS
═══════════════════════════════════════════════════════════════════════════════════════

1. NE JAMAIS APPROCHER LA MAIRIE SEUL
   Toujours accompagne du syndicat. Le syndicat est ton porte-parole.

2. NE PAS PARLER DE TAXATION AU PREMIER CONTACT
   Parler de "aide", "gagner plus", "optimisation". La taxation vient APRES.

3. L'APP DOIT ETRE INDISPENSABLE POUR LE CHAUFFEUR
   Il doit gagner PLUS qu'il ne paie. Sinon, il desinstalle.

4. OFFLINE FIRST
   L'app doit fonctionner sans internet. C'est non negociable en Afrique.

5. TRANSPARENCE TOTALE
   Avec le syndicat, avec la mairie, avec les chauffeurs. Pas de secrets.

6. PATIENCE
   Ca prend 6-12 mois pour etre rentable. C'est un marathon, pas un sprint.
